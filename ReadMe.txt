Thank you for downloading AuriaOS!
AuriaOS is an Operating System designed on PowerPoint.

Before you get started, do the following steps:
1. Extract the .zip file as a folder with the name "AuriaOS".
2. Open the "AuriaOS.ppsm" file to get started on using the OS!

AuriaOS uses PP Radio Grotesk (Light, Regular and Bold), and PP Right Sans. If you don't have those fonts, Don't worry! You can install them on the "Fonts" folder!

Also, Make sure you have macros always enabled. If not, do the following steps:

For users with PowerPoint 2000, 2002 or 2003:
1. Select Tools | Macro | Security.
2. In the Security Level page, select Medium.
3. Click OK to dismiss the dialog box.

For users with PowerPoint 2007:
1. Select the Office Button or press Alt+F key combination.
2. Click on PowerPoint Options.
3. Select Trust Center in the left pane.
4. Click Trust Center Settings button in the right pane.
5. Select Macro Settings in the left pane.
6. Select Disable all macros with notification option.
7. Click the OK button.

For users with PowerPoint 2010 or later:
1. Select the File menu or press Alt+F key combination.
2. Click the Options button in the left pane.
3. Select Trust Center in the left pane.
4. Click the Trust Center Settings button in the right pane.
5. Select Macro Settings in the left pane.
6. Select Disable all macros with notification option.
7. Click the OK button.
